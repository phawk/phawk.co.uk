# phawk.co.uk

This is my personal site, powered by [Gatsby](https://www.gatsbyjs.org/).

## Developing

```sh
$ gatsby test
$ gatsby develop
```

## Deploy

Deploys are handled by [Netlify](https://www.netlify.com/) and happen automatically.

## Editing

### Creating a post

```sh
touch ./content/blog/2020-06-09-new-post-title.md
```
